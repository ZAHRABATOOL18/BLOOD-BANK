# Blood Path — Blood Bank Management System

Complete enterprise blood donation management system 
built with Oracle APEX and PL/SQL.

## Tech Stack
- Oracle APEX 26.1
- PL/SQL
- Oracle Database
- ORDS REST APIs
- SQL

## Features
- Custom authentication with role-based access (4 roles)
- 10 entity database — fully normalized
- 30 pages — CRUD, reports, dashboard, global search
- Real-time analytics with charts
- Donor eligibility auto-tracking
- Blood unit expiry alerts
- Camp scheduling and participation tracking
- Request fulfillment workflow
- Form validations on all inputs
- REST APIs for external integration

## Entities
Donor, Blood Bank, Blood Unit, Hospital, 
Request, Donation Camp, Camp Participation, 
Fulfillment, Staff, Patient
